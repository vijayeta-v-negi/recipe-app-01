# Recipe CRUD API

A simple Recipe CRUD API built using Node.js, Express.js, and MongoDB following the MVC architecture.

## Features

- Create Recipe
- Get All Recipes
- Get Recipe By ID
- Update Recipe
- Delete Recipe
- MongoDB Database Connection
- MVC Pattern
- REST API
- Postman Tested


## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon
